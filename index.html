V.E.R.A Guidance

How to install the desktop app and a list of available commands.

How to Install V.E.R.A Desktop

Follow these steps to get the V.E.R.A desktop version running on your computer.

  1. Download the correct version for your operating system (Windows, Mac, or Linux) from the Download section.
  2. Unzip the downloaded file to a folder you can easily access.
  3. Run the `V.E.R.A.exe` (on Windows) or the corresponding executable for your system.
  4. Follow any on-screen prompts to install dependencies (like Python, if not already installed).
  5. Once installed, launch V.E.R.A from your desktop or applications folder.

V.E.R.A Command List

Here are some of the commands V.E.R.A understands:

  • hello / hi / hey - Greet V.E.R.A.
  • how are you - Check on V.E.R.A.
  • what time is it - Get the current time.
  • what is the date - Get today's date.
  • what is 25 * 17 - Perform math calculations.
  • open google.com - Open any website in a new tab.
  • open youtube - Open common websites by name.
  • (Paste a YouTube URL) - V.E.R.A will offer to download the video as MP3 or MP4.
  • tell me about... - Ask any general knowledge question for a web search.
